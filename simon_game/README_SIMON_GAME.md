# SIMON GAME
 ## HOW TO PLAY?
<b> STEP-1</b>: PRESS ANY KEY TO START THE GAME.<br>
 <b>STEP-2 </b>: LEVEL-1 WILL START. ONE OF THE FOUR COLOUR BUTTONS WILL BLINK. USER NEEDS TO REMEMBER AND CLICK THE SAME BUTTON.<br>
 <b>STEP-3</b>: LEVEL-2 STARTS. ANOTHER BUTTON WILL BLINK. REMEMBER THIS AND THE LAST ONE TOO AND CLICK IN SEQUENCE: FIRST THE BUTTON WHICH BLINKED BEFORE AND THEN THE BUTTON WHICH BLINKED NOW.
 <br><b>STEP-4</b>: LIKE THIS LEVELS WILL COME. REMEMBER THE BUTTON PSTTERN FROM LEVEL 1 AND CLICK THEM IN SEQUENCE WHICH WILL INCREASE LEVEL BY LEVEL.
 WHEN YOU FORGOT AND CLICK THE WRONG ON THE GAME WILL RESET.<br>
